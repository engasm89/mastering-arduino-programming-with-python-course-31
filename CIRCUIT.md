## Connections

- Onboard LED at `D13` (no external wiring required).
- USB cable to PC for serial.

## Diagram (ASCII)

```
 Arduino ---- USB ---- PC (Python)
  D13 LED onboard
```

## Notes

- Ensure the correct COM port and matching baud rate (`115200`).
- Commands must end with newline `\n` or carriage return `\r`.
